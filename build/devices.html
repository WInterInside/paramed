Металлодетектор VMF4

Характеристики

Четвертое поколение: компактный металлодетектор со сменной поисковой рамкой для проектов по разминированию

  • Оптимизированная эргономичность для длительного применения в гуманитарном и военном разминировании
  • Складной металлодетектор с малыми габаритами Изменяющаяся оптимизация применения при помощи сменных поисковых рамок VALLON-UXO-программное обеспечение (для неразорвав-шихся боеприпасов) в сочетании с поисковыми рамками для неразорвавшихся боеприпасов (UXO) VS30 и VS60
  • Обнаружение материалов и проволок, слабо проводящих электричество
  • Клавиша «Pinpoint» для более высокой точности местонахождения
  • Интегрированная GNSS-система для автоматического и ручного сохранения путевых точек
  • Инфракрасный светодиодный индикатор для использова-ния с приборами ночного видения и вибросигналом Bluetooth®и Wi-Fi - разъем к внутреннему веб-серверу для индивидуальных настроек прибораОптимизированная эргономичность для длительного применения в гуманитарном и военном разминировании
